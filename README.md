# B-2 Spirit Composite Model Website

This package contains a small multi-page static website for GitHub Pages.

## Files
- `index.html` - home page
- `fabrication.html` - manufacturing process
- `repair.html` - repair and spare-model work
- `gallery.html` - photo gallery
- `build-guide.html` - beginner step-by-step website guide
- `style.css` - shared styling for every page
- `assets/` - project images used by the pages

## Publish on GitHub Pages
1. Create a repository.
2. Upload all files so `index.html` is at the top level of the repo.
3. Upload the `assets` folder too.
4. In GitHub, open **Settings > Pages**.
5. Publish from the `main` branch and the root folder.

GitHub Pages hosts static files directly from a repository and can publish from a branch/folder source. See GitHub Docs for current steps.
